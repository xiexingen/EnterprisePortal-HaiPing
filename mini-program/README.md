# 报税系统
### 目录结构
```
+tax/
+----            ------config配配文件
+----            ------dist编译后的可执行文件
+--src/          ------common配置请求地址的公共文件
+--src/          ------images存放图片
+--src/          ------pages存放页面
+--src/          ------service存放请求api
+--src/          ------utils存放一些公共方法
```
### 下载依赖
```
npm install 或 cnpm install 
```
### 启动
```
npm start 或yarn start
```
